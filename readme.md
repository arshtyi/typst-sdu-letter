# typst sdu letter

The typst template for Shandong University letterhead.

- Please carefully consider whether to use this template, any consequences are not related to the author.
- This letterhead is not unified in reality (see below for details). Only one version is implemented.

> 使用typst制作的山大信纸模板。
>
> - 请谨慎考虑是否使用此模板，任何后果与作者无关。
> - 此信纸现实中也并未统一（详见下文）。仅实现了一个版本。

## Usage

There are no more settings exposed, please consider modifying the source code if needed. See `template.typ` for an example.

## Style

The different versions of the letterhead look different, including but not limited to:

- Size
- Margin
- Logo (position, size, layout, color, font)
- Footer (position, color, font)
- Horizontal line (color, position, thickness)

## Note

This does not fully reproduce handwriting, but it can be improved by using a more handwriting-like font, such as [zi2zi-JiT](https://github.com/kaonashi-tyc/zi2zi-JiT) and introducing some jitter.
